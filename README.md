# 微信创业模拟器 (WeChat Startup Simulator)

> 🎮 一款模拟微信产品决策与项目管理的策略游戏

## 📖 项目概述

**微信创业模拟器**让玩家扮演微信团队负责人，从2011年南通6楼的"小黑屋"开始，经历30轮关键决策，体验微信从0到14亿用户的完整旅程。

每一个选择都会影响产品的命运——用户增长、团队士气、公司资源、产品口碑...你能带领微信走向传奇吗？

## 🎯 核心特色

- **30轮历史事件**：基于真实微信发展史改编
- **六大核心数值**：用户、口碑、创新、营收、团队、资源
- **延迟反馈机制**：选择的后果可能在2-5轮后才显现
- **多重结局**：从传奇到裁撤，6种以上不同结局
- **两难选择**：没有明显正确答案，考验你的判断力

## 💾 数值系统

```
👥 用户数    - 产品的用户规模
⭐ 口碑度    - 用户信任与满意度（0-100）
💡 创新力    - 团队突破能力（0-100）
💰 营收      - 给公司创造的收入
👨‍💻 团队      - 人才密度与士气（0-100）
🔋 资源      - 公司给的支持（0-100）
```

## 🎬 结局类型

- 🏆 **传奇**：成为国民级应用
- 🥇 **成功**：成为一线产品
- 🥈 **及格**：成为有影响力的产品
- 🥉 **存活**：勉强活下来
- 💀 **项目裁撤**：公司不再支持
- 💀 **团队解散**：核心成员离开
- 💀 **被合并**：并入其他团队
- 💀 **边缘化**：成为养老项目
