## 微信七大价值观（龙七条）

1. **做对用户有价值的事情** - 哪怕舍弃短期利益
2. **保持自身的价值观** - 价值观体现在产品中，不做双面人
3. **保持小团队，保持敏捷** - 避免官僚化，只招最优秀的人
4. **学习和快速迭代比经验更重要** - 面向新环境产生思路
5. **系统思维** - 建造透明公正的自运转系统，避免人为干预
6. **让用户带来用户，口碑赢得口碑** - 网状传递效应
7. **思辨胜于执行** - 提倡争论，通过辩理找到正确方法
